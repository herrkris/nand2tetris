# Nand Gate

Returns 0 when both inputs are 1, otherwise 1.

## Api

```
Input: a, b
Output: out
```

## Truth table

| a | b | out |
|---|---|-----|
| 0 | 0 |  1  |
| 1 | 0 |  1  |
| 0 | 1 |  1  |
| 1 | 1 |  0  |

## Implementation

We consider the `Nand` gate to be a primitive and therefore we do not need to implement it.
